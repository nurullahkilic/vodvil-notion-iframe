# Vodvil Notion Iframe

An iframe for the [vodvil.app](https://vodvil.app) project. Built with React, Zustand, Emotion and Flask.


## Basic Usage

```html
<iframe 
   src="https://embed.vodvil.app/?username=[VODVIL_USERNAME]"
   width="400" 
   height="652"
   style="border-radius:12px" 
   frameBorder="0" 
   loading="lazy">
</iframe>
```
